Projekt: Expense Tracker (Tracker Wydatków)
Opis projektu
Aplikacja do śledzenia osobistych wydatków z kategoriami, datami i możliwością przeglądania statystyk.
Funkcjonalności:

Dodawanie wydatków (kwota, kategoria, opis, data)
Wyświetlanie listy wydatków
Usuwanie wydatków
Filtrowanie po kategorii
Podsumowanie wydatków (suma ogólna, suma per kategoria)

Stack technologiczny:

Backend: Node.js + Hono
Database: SQLite (plik databases/pra{x}.db dla każdego z was)
Frontend: HTML + CSS + Vanilla JavaScript
